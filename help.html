How to setup Website

Setup GitHub.

You will need a Github Account. Once you set it up, tell me your username and I can give you access to editing the website

Download Git

Download Git from here. You can use all the default settings when installing git

Cloning the repo

First, create the folder where you want to store the project files. Inside the folder, right click and click on the "open in terminal" button.
Then in your teriminal, use the following commands:

git init
git remote add origin https://github.com/Jeremy924/Jeremy924.github.io.git
git pull origin main
git checkout main

Editing the HTML

Just copy the formatting for the other html to add whatever you want. Ask ChatGPT to add images to the HTML.
To make the images accessible, just put them into the project folder. For your resume, you can also just put it in the folder, just make sure to change the link to be the name of the file.
Make sure you put everything in the /joshua folder within the project folder, otherwise it won't find any of the files

Viewing the Website Locally

Open up your web browser, and type file:/// followed by the full file address for the project folder (ex. C:\Users\Jerem\...). Add /Joshua to view your portfolio

Pushing the Changes to the Public Website

Go back to the terminal, make sure it is in the project folder. Run the following commands. Make sure you update the message for git commit so that it is something meaninful

git add .
git commit -m 'message'
git push origin main



Wait a few seconds for it to send it to gitub, then go back to the public website and it should now be visible (you may have to wait a few minutes for the website to actually update)
